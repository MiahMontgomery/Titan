# Titan: AI-Powered Project Management System

Titan is an intelligent, locally-deployable project management system powered by AI agents designed to revolutionize software development workflows, offering advanced collaboration and optimization tools.

## Core Technologies

- TypeScript React frontend
- Node.js backend
- WebSocket real-time communication
- Firebase integration
- Modular AI agent architecture
- OpenAI API connectivity (GPT-4o integrated)
- Browser automation with Puppeteer
- Web accounts management system
- LowDB for data persistence

## Features

- AI-assisted project management
- Autonomous task scheduling and execution
- Browser automation for web-based tasks
- Real-time communication with WebSockets
- Persona management with isolated containers
- Progress tracking with milestones and goals
- Performance metrics and visualization

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- OpenAI API key for AI functionality
- Firebase credentials (optional, for cloud syncing)

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/MiahMontgomery/Titan.git
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Create a `.env` file with your API keys:
   ```
   OPENAI_API_KEY=your_openai_api_key
   FIREBASE_API_KEY=your_firebase_api_key (optional)
   FIREBASE_APP_ID=your_firebase_app_id (optional)
   FIREBASE_PROJECT_ID=your_firebase_project_id (optional)
   ```

4. Start the development server:
   ```
   npm run dev
   ```
